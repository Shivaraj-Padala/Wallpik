# Wallpik the stock photos & wallpaper web app ([Preview](https://shivaraj-padala.github.io/Wallpik/index.html))
<img src="/assets/favicon.png" width="80" /> <br />
Wallpik is a single page application built with HTML, CSS, Vanilla JavaScript & [Pexels API](https://www.pexels.com/api/) which lets you browse and download from millions of stock photos and wallpapers.
