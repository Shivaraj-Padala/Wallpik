Wallpik is a single page application built with HTML, CSS, Vanilla JavaScript & Pexels API which lets you browse and download from millions of stock photos and wallpapers.<br /><br />

## 🚀 Getting Started
- Download or clone the repo.
- Get your free [Pexels API](https://www.pexels.com/api/) key.
- Replace or copy paste your API key in **AUTH_HEADER** which is located in **main.js** file.<br /><br />

For more information about Pexels API refer to [documentation.](https://www.pexels.com/api/)<br /><br />
